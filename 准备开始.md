###(2).前端框架与库的区别
1. jquery 库 -> DOM(操作DOM) + 请求
2. art-template库 -> 模板引擎
3. 框架 = 全方位功能齐全
  简易的DOM体验 + 发请求 + 模板引擎 + 路由功能
4. KFC的世界里，库就是一个小套餐，框架就是全家桶
5. 代码上的不同
  一般使用库的代码，是调用某个函数，我们自己把控库的代码
  一个使用框架，其框架在帮我们允许我们编写好的代码
    框架：初始化自身的一些行为
      执行你所编写的代码
      拖放一些资源
